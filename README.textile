h1. Top4R

Top4R 封装了 "淘宝开放平台":http://open.taobao.com 的 Ruby 接口，帮助你快速构建基于TOP的应用程序。

h2. 功能

* [09/25/10] TOP API 2.0支持
* 商品
* 物流
* 店铺
* 收费
* 淘宝客
* 交易
* 用户

h2. 安装

	$ gem install top4r
